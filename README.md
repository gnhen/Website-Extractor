# Website-Extractor
Pulls ALL files from a website.

# Usage

## Option 1
### Using EXE Package
1. Download EXE File from <a href="https://github.com/gnhen/Website-Extractor/releases">Releases</a>
2. Run EXE File
Done!

## Option 2
### Running Pure Python Code
1. Download
2. Open Command Line
3. Run ```python app.py```
Done!
